BOSCH'S TRAFFIC SIGN RECOGNITION

Inter IIT Tech Meet 2021

Team Code : H1_BSC_2

Features

Addition of image to dataset

The user can add an image to any of the 48 (43 original + 5 newly created classes) from his computer/phone in the form of a jpeg/png format.

Image augmentation features during addition

The user can while adding an image will have the option of blurring the image using pixel dimension, rotating the image to any extent he/she feels necessary and also the option of cropping the image.

Visualizing the dataset

The user will have the option of randomly visualizing a part of the dataset or viewing a set of images from a particular class too.

Testing the new dataset

If the user wants he can test our pretrained model on the updated dataset which will include the images he/she added to the dataset.

Visualization of results

The user can then choose any of the following options:

  • Get the accuracy value for a particular class
  • Get the entire f1 matrix displaying precision, recall and f1 score for each class separately
  • View the cases in which the model in failing to predict the right class
  • View the misclassified samples
  • View the per class frequency
